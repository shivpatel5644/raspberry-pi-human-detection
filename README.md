# Real-Time Human Detection & Tracking (Raspberry Pi)

## 📌 Project Overview
This project implements a real-time human detection and tracking system using a Raspberry Pi. The system uses computer vision techniques to detect human presence, track movement, and generate positional decisions in real time.

## 🚀 Features
- Real-time human detection using YOLO
- Object tracking system
- Edge deployment on Raspberry Pi
- Performance evaluation (FPS, latency)
- Experimental testing under different conditions

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO (ONNX)
- Raspberry Pi
